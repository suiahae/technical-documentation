### 创建本地安装缓存

```
vs_community.exe --layout c:\vs2019layout --add Microsoft.VisualStudio.Workload.NativeDesktop;includeRecommended --add Microsoft.VisualStudio.Workload.Python –-add Component.GitHub.VisualStudio –-add Microsoft.VisualStudio.Component.Git --lang zh-CN
```

### 从本地缓存安装 Visual Studio

```
c:\vs2019layout\vs_community.exe  --add Microsoft.VisualStudio.Workload.NativeDesktop;includeRecommended --add Microsoft.VisualStudio.Workload.Python –-add Component.GitHub.VisualStudio –-add Microsoft.VisualStudio.Component.Git
```

## Win10 利用安装盘启用 .NET Framework 3.5
以管理员身份运行命令提示符，在“管理员:命令提示符”窗口中输入以下命令：
```
dism.exe /online /enable-feature /featurename:netfx3 /Source:D:\sources\sxs
```
注：其中的盘符D要改成你实际的虚拟光驱盘符
